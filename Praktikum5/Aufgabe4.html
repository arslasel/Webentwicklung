Vergleich zwischen Object.create und Object.beget Why Beget? Becuase this has everything to do with the new keyword. In JavaScript, new only works with functions (which are a special type of object). The type of object you get back depends on that function's prototype object
Statt vom prototyp Object sind die Objekte von beget vom Prototyp F (macht Sinn weil der Prototyp F verwendet wurde)
